![Cover](/src/main/resources/static/images/cover/cover1.png)

##  專案介紹

&ensp;&ensp;「**Time To Travel**」，是一個讓無論是有訂房需求的使用者，或者具有特色民宿、飯店的業者，都能夠滿足自己需求的訂房網站。  
&ensp;&ensp;此專案為緯育 TibaMe - Java 雲端服務開發技術養成班期末發表專案，小組成員共八人，各自完成分配到的功能之前後端所有部分。  
&ensp;&ensp;我所負責的功能為:
    1. **房型搜尋、景點搜尋**
    2. **房型下訂**
    3. **消費者訂房紀錄查看**
    4. **消費者評論新增與修改**
    5. **前端公版頁面**
